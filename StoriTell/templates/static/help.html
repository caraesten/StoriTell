Help out!


StoriTell can't grow without your help. If you like using the site and want to keep seeing quality stories, please spread the word!


How to help

There are two ways you can help expand this project:

First: You can print out a PDF poster from this page and put it wherever you think the most people are likely to see it. The text comes from one of the most popular stories on the site.

Second: You can take one of the existing posters and change the design or story on it. All of these posters were made using the open-source design software Scribus, which means anyone can download the software and make changes. All we at StoriTell ask: Keep the StoriTell logo and a link to the site.
